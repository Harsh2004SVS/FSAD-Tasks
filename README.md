## How To Run

### Tasks 1 to 6 (PHP + SQLite)

1. Install PHP (8+ recommended).
2. Open terminal in `c:\MANVITH\FSAD`.
3. Run:

```powershell
php -S localhost:8000
```

4. Open in browser:

- `http://localhost:8000/Task1_Student_Registration/index.php`
- `http://localhost:8000/Task2_Data_Dashboard/index.php`
- `http://localhost:8000/Task3_Login_Validation/index.php`
- `http://localhost:8000/Task4_Order_Management_Joins/index.php`
- `http://localhost:8000/Task5_Transaction_Payment/index.php`
- `http://localhost:8000/Task6_Triggers_Views_Logging/index.php`

### Task 7 (Static HTML/CSS/JS)

Open:

- `Task7_Interactive_Form/index.html`

(or access via local server: `http://localhost:8000/Task7_Interactive_Form/index.html`)
